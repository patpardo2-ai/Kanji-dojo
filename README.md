# 漢字道場 · Kanji Dojo

App de estudio de kanji japonés JLPT N4-N2 con mascota virtual.

## 🌐 Acceder a la app

👉 **https://TU-USUARIO.github.io/kanji-dojo/**

## ✨ Características

- 🐰 Mascota virtual Haru (conejo) que se alimenta con tus aciertos
- 🃏 Flashcards N4/N3/N2 con palabras compuestas
- 📝 Examen estilo JLPT con opciones A·B·C·D
- 📖 Textos de lectura con furigana
- 📋 Lista completa de kanji N4 (190 kanji)
- ❌ Mis fallos / ✅ Ya vistos
- 💾 Progreso guardado en el dispositivo
- 📱 Instalable como app (PWA)

## 📱 Instalar como app

**iPhone/iPad:** Safari → Compartir ⬆️ → "Añadir a pantalla de inicio"  
**Android:** Chrome → menú ⋮ → "Añadir a pantalla de inicio"
